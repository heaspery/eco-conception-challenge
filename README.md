# Éco-conception Challenge

Ce dépôt présente le projet d’éco-conception réalisé dans le cadre du cours **"Éco-conception Challenge"** dispensé à la HEIG-VD.  
L’objectif était de réaliser la page **"Play & Win"** du site des Jeux Olympiques **Milano Cortina 2026**, dans une approche éco-conçue.

## Itérations

1. **Version 1** : https://eco1.ancay.ch  
   Cette version a été pensée pour satisfaire au mieux le but ludique de la page et les besoins promotionnels des sponsors.

2. **Version 2** : https://eco1.ancay.ch/v2  
   Cette version a été pensée dans le même but, dans une version encore plus simplifiée.

## Technologies
Ce projet a été conçu avec Nuxt, Nuxt Image ainsi que Tailwind CSS.

## Lancer le projet

Après avoir cloné le projet et installé les dépendances, vous pouvez le lancer en mode développement avec la commande :

`npm run dev`

Pour générer le site statique, vous pouvez utiliser la commande :

`npm run generate`
